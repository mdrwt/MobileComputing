Assignment3 to demo various storage options available in android.

KeyValue: Screen shows two values corresponding to two keys in Activity. If no value is present only hint text is shown. On tapping edit option user can edit textfield 
to add text and use 'SAVE' button to save value to corresponding keys.

Storage: User can enter filename and content to save file to storage or enter only filename to get file from storage. User can also save to/get from private external storage by 
checking the private checkbox.

SQLITE: User can tap menu option to add new student record. User can tap on an existing user record in list to move to edit screen. Here user can either delete the record by 
tapping 'DELETE' button or update record by tapping 'UPDATE' button. 
NOTE: Here UPATE button is used both for update and save functionality.
